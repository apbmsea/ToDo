# Тестовое задание по реализации ToDo приложения для Маркетингово агенства Victory 

## Запуск проекта

- ###  Установка зависимостей 

```bash
npm install
```
/

```bash
yarn
```

- ### Инициализация
```bash
npm run dev
```
/

```bash
yarn dev
```
